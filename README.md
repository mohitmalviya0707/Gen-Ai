# 🤖 AI Mood Chatbot

A simple AI chatbot that changes its response style based on selected mood (Angry, Funny, Sad). Built using LangChain, Mistral AI, and Streamlit.

---

## Features

* Mood-based responses (Angry / Funny / Sad)
* Interactive chat interface
* Conversation memory support
* Clean and lightweight UI

---

## Tech Stack

* Python
* Streamlit
* LangChain
* Mistral AI
* HuggingFace

---

## Setup

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
pip install -r requirements.txt
```

Create a `.env` file:

```
MISTRAL_API_KEY=your_key
HUGGINGFACEHUB_API_TOKEN=your_token
```

Run the app:

```bash
streamlit run app.py
```

---

## Usage

* Select a mood
* Start chatting
* Type `0` to stop
* Use reset button to restart

---

## Notes

* `.env` file is ignored for security
* API-based models recommended for low-end systems
